Sierpinski’s Triangle Generator

Sierpinski’s Triangle Generator (C#) utilizes the graphics class and recursion to generate the fractal using 2 methods. One method involves the user manually inputting the X and Y values. The other method, involves the user using the mouse to select 3 random points on the canvas per triangle.

Simple Scientific Calculator

The Simple Scientific Calculator is the first C# project I came up with. It completes many tasks including calculating arithmetic average, median, mode, minimum value, nth smallest value and randomizes the list order from inputted values.

As well, this was my first introduction to user interface and design. I used data grid view and text labels and implemented them creatively for visual satisfication.

Sodoku Solver

Sodoku Solver was made using python's pygame library and can solve any sodoku board. However, user functionality is currently limited. I will be working on this in the future to make it more user friendly.

Furthermore, the sodoku solver taught me the importance of backtracking, an algorithm that becomes useful when solving difficult coding problems.

To-do List

During my school time in university I found it really difficult to keep track of all my classes. As a result, I made a simple to-do list to keep track of my course work. This project help me get comfortable with javascript and taught me the importance and implementation of local storage.
